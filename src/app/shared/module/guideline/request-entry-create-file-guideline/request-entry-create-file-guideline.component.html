Form filling guidelines

It is important to read the following guidelines to be able to successfully fill in and submit the form.

  • A Student is only allowed to enter a valid email address.
  • The payment will only be valid if the student have the paid amount required to fulfill the request.
  • Any attempt to circumvent the process or breach or modify pre-defined system rules is prohibited.
  • Only Images are allowed to be uploaded in the form input fields.
  • Images that are to be uploaded should not exceed 500 kilobytes in size.
  • Other Form inputs not allowed by this form will be deemed invalid for processing.
  • Multiple Image upload for each input field is not allowed.
  • Only {{fileType}} documents are allowed to be uploaded in the form input fields.
  • {{fileType}} documents that are to be uploaded must not exceed 500 kilobyte in size.
  • Other Form inputs not requested by this form will be deemed invalid.
  • Multiple {{fileType}} documents upload for each field are not allowed.
  • Only valid input are allowed in the form fields.
